# FlockSim
A flock simulation written in python

Controls:

Arrowkeys 
  left: rewind
  right: fast forward
space: play/pausu
- : zoom out
+ : zoom in
r: reset
n: new simulation
F2: show fps
F3: debug mode
esc: close application

mouse left click on screen: create individual. Hold to set velocity
mouse right click: move screen
scroll wheel: zoom in/out
